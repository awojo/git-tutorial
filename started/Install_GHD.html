Installing GitHub Desktop

The site automatically recognizes your operating system and prompts you with the Git software download that you will require.

Click on the download button on the GitHub Desktop page: GitHub Desktop Download and open the install package from the destination folder.

If you encounter any install errors on macOS, please reference the workaround mentioned on our page which walked you through the install for Git: Git install for mac OS
