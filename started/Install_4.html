GitHub Account and Desktop Installation

GitHub Homepage

To create your GitHub account, check out these steps before heading over to the GitHub home page.

The GitHub Home Page

In the upper right corner, click on the Sign Up button. Follow the prompts to enter your email address, create a Password, and create a User Name:

Continue following the prompts through the verify steps:

Once your account is successfully created, you just need to Sign in:

Upon signing in, you will be directed to your profile where you can further fill out profile data, find your repositories and projects, find notifications, follow your contacts, and update other settings as desired:

Installing GitHub Desktop

The site automatically recognizes your operating system and prompts you with the Git software download that you will require.

Click on the download button on the GitHub Desktop page: GitHub Desktop Download and open the install package from the destination folder.

If you encounter any install errors on macOS, please reference the workaround mentioned on our page which walked you through the install for Git: Git install for mac OS
