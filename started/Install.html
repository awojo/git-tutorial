Getting Git and GitHub Software Installed

Here is a checklist of items you will need to complete to get started using Git, GitHub, and GitHub Desktop:

1 – Choose an Installer Package and Install the Git software Get instructions from the Git site
2 – Create your GitHub Account: Choose Username and Password Go to the GitHub Website
3 – Install the GitHub Desktop Application GitHub Desktop Install Page
4 – Complete your first log-in at GitHub and Authenticate your account GitHub Log-in Page
5 – Configure GitHub Desktop basic settings so that you can contribute to projects GitHub Projects Quickstart
6 – Update your GitHub profile and customize GitHub Desktop GitHub Profile Settings
